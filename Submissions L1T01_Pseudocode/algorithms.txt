2.2

Pseudo code solution:

* Request user to input their age
* Repeat request if user inputs anything other than integers
* Store input in variable called age
* Create variable called currentYear
* Assign currentYear to the current year (eg. currentYear = 2023)
* Output "Based on your age of [age] , it looks like you were born in [currentYear - age]."
